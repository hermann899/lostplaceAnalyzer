Satellitenbild + Struktur-Detektor (Single-HTML)

Analyse läuft offline im Browser entweder direkt aus der aktuellen Kartenansicht (wenn der Tile-Server CORS erlaubt) oder auf einem Bild, das du hochlädst. Danach kannst du Ergebnisse mit Ja/Nein labeln, wodurch ein einfacher Lernmechanismus (kNN) besser wird.

1) Bildquelle für Analyse

Du kannst auch „Analyse aus Kartenansicht“ nutzen. Falls das wegen CORS blockiert ist, nimm Upload (Screenshot/Export).

2) Detektor-Einstellungen

Die Erkennung ist heuristisch (klassische CV) + kNN-Feedback. Für „echtes“ Deep Learning bräuchte man ein trainiertes Modell + passenden Datensatz.

3) Ergebnisse + Feedback (lernend)

Klicke ein Ergebnis an, bewerte es mit Ja/Nein. Das speichert Feature-Vektoren lokal und verbessert die automatische Einstufung.

Info

- Karte nutzt eine frei verfügbare Satelliten-Kachelquelle (nur Anzeige).
- Analyse läuft auf Upload-Bild, damit Canvas nicht „tainted“ wird.
- Speicherung: LocalStorage + Export/Import JSON (Download/Datei-Auswahl).
- Datenschutz: Alles bleibt lokal in deinem Browser.
